A collection of personal tools and utilities.

## Tools

| Tool | Description | Vibecoded |
|------|-------------|:---------:|
| [font-to-png](font-tools/font-to-png/README.md) | Export every glyph in a `.ttf`/`.otf` font to individual transparent PNG files. Supports SMuFL fonts, dynamic sizing, uniform scale mode, and whitelist filtering. | ✅ |
